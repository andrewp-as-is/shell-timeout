### Installation
```bash
$ [sudo] pip install shell-timeout
```

```bash
$ [sudo] npm i -g shell-timeout
```

#### How it works
If the command times out, then exit with status 124. Otherwise, exit with the status of COMMAND

#### Examples
```bash
$ timeout 1 sleep 2
```
