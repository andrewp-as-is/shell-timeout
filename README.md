#### How it works
If the command times out, then exit with status 124. Otherwise, exit with the status of COMMAND

#### Scripts usage
command|`usage`
-|-
`timeout` |`usage: timeout seconds command ...`

#### Examples
```bash
$ timeout 1 sleep 2
```
